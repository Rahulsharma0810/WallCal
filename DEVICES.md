# Supported Wallpapers

| Platform | Script | Resolution | Output |
| --- | --- | --- | --- |
| iPhone | `generators/iphone.js` | `1290x2796` | `images/iphone-wallpaper.png` |
| iPad Air 11" (Portrait) | `generators/ipad-portrait.js` | `1640x2360` | `images/ipad-air-11-wallpaper.png` |
| iPad Air 11" (Landscape) | `generators/ipad-landscape.js` | `2360x1640` | `images/ipad-air-11-wallpaper-landscape.png` |
| Mac | `generators/mac.js` | `3456x2234` | `images/mac-wallpaper.png` |

## Local Commands

- `npm run generate:iphone`
- `npm run generate:ipad`
- `npm run generate:ipad-landscape`
- `npm run generate:mac`
- `npm run generate:all`
- `npm run verify`
